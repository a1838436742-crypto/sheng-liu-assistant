# 景别类型

网页内可见的景别标签必须使用中文。内部筛选代码可以保留英文缩写，方便 CSS 和 JS 稳定工作。

## 映射表

| 内部代码 | 中文可见标签 | 样式类 | 用途 |
|---|---|---|---|
| `WS` | 远景 / 全景 | `p-ws` | 建立空间、全身、地点信息重 |
| `MS` | 中景 | `p-ms` | 半身、对白、双人镜头 |
| `CU` | 近景 / 特写 | `p-cu` | 头肩、反应、情绪 |
| `ECU` | 大特写 | `p-ecu` | 眼睛、手、道具、屏幕 |
| `MACRO` | 微距 | `p-macro` | 毛孔、水滴、织物、细节纹理 |
| `PAN` | 摇镜 | `p-pan` | 水平或垂直缓慢扫过 |
| `OS` | 画外声 | `p-os` | 声音提示，无明显画面变化 |
| `VO` | 旁白 | `p-vo` | 旁白覆盖当前画面 |
| `VO+MS` | 旁白 + 中景 | `p-vo` | 中景画面中叠加旁白 |
| `DISSOLVE` | 叠化 | `p-dis` | 场景或时间过渡 |

## HTML 用法

`data-plan` 使用内部代码，徽章文字显示中文：

```html
<tr data-scene="21" data-plan="ECU">
  <td class="c-num">21.4</td>
  <td class="c-plan"><span class="badge p-ecu">大特写</span></td>
  ...
</tr>
```

筛选下拉框同样使用内部代码作为值，中文作为用户可见文字：

```html
<option value="ECU">大特写</option>
```

## 不要随意新增代码

如果镜头不好归类，选择最接近的现有代码。更细的差异写进“摄影机”栏，例如“缓慢推进”“手持跟拍”“从门内向外拍”。

## 摄影机栏

摄影机栏必须使用中文。常见写法：

- 固定机位
- 手持
- 广角 / 固定
- 缓慢推进
- 缓慢拉远
- 左至右轨道
- 右至左轨道
- 缓慢摇镜
- 升降镜头
- 正上方俯拍冻结
- 单镜头，50mm
- 门内视角手持
- `—` 用于叠化等过渡行
