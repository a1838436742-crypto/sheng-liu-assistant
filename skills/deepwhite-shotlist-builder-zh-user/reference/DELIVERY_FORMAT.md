# 交付形式确认

最终输出前必须询问用户是否需要生成网页。不要默认生成 HTML。

## 必须询问

在资产、位置、时间划分、提示词结构都确认后，问：

```text
请确认最终交付形式：
A. `生成网页`：输出完整中文 HTML 分镜提示词表，包含时间安排、镜头分解、提示词复制区。
B. `只要文字提示词`：不生成网页，直接按顺序输出每条视频提示词。

请回复：`生成网页` 或 `只要文字提示词`。
```

## 如果用户选择生成网页

- 使用 `templates/HTML_TEMPLATE.md`
- 保留时间安排列、镜头分解、提示词复制区
- 运行 `reference/HTML_QA.md`
- 输出 HTML 文件

## 如果用户选择只要文字提示词

- 不生成 HTML
- 不运行 HTML QA
- 直接在聊天中输出：

```markdown
## 提示词 1｜场景内 00:00-00:08｜约8秒

```text
[完整视频提示词]
```

## 提示词 2｜场景内 00:08-00:15｜约7秒

```text
[完整视频提示词]
```
```

- 每条提示词仍必须遵守 `PROMPT_PATTERNS.md`
- 如果有故事板，仍要写故事板编号和禁止标注进入画面的规则

## 变更规则

如果用户在生成过程中改变交付形式，回到本 Gate 重新确认。
