# 摄影机与情绪同步

摄影机是焦点人物情绪的外化。镜头运动、焦段、持续时间必须由人物心理状态决定，而不是泛泛追求“电影感”。

## 1. 情绪到摄影机运动

| 焦点人物情绪 | 摄影机类型 | 中文写法 |
|---|---|---|
| 愤怒 / 暴怒 / 紧张 / 边缘状态 | 不安手持，节奏破碎，小幅不规则晃动 | `摄影机不安、紧张——画面有较明显的呼吸式漂移、上下微抖、左右轻摆。手持摄影，节奏不规则。禁稳定器。` |
| 冷静 / 掌控 / 自信 | 克制手持，规律细微呼吸感 | `摄影机平稳、流畅、克制——只剩极细微、规律的呼吸式微动。手持摄影但呼吸感极弱。` |
| 悲伤 / 脆弱 | 低位缓慢手持，呼吸频率降低，轻微下沉 | `摄影机缓慢、低位手持——呼吸节奏放慢，画面有极轻微下沉感。` |
| 震惊 / 揭示 | 开始静止，随后极慢推进或拉远 | `镜头开始严格静止——0.5秒后开始极缓慢推进/拉远，整段移动距离不超过15厘米。` |
| 动作 | 清晰运动，合理快门运动模糊 | `60fps流畅运动，180°快门运动模糊，禁止超出快门范围的拖影。` |
| 最终判决 / 定格节拍 | 正上方俯拍冻结 | `严格正上方俯拍。0.3-0.5秒冻结。所有人物位置冻结。时间静止。` |

## 2. 单镜头内的情绪变化

如果同一个连续镜头里情绪发生变化，要分阶段写摄影机变化，并和表演编号绑定：

```text
⚠️镜头开始（步骤①②，Roko处于怒气）：摄影机明显手持、不安、紧张，画面有呼吸式漂移、上下微抖、左右轻摆。

⚠️随着Roko情绪逐步缓和（步骤③④⑤）：摄影机的不安逐渐减弱，漂移幅度变小，左右晃动减少，画面开始稳定。

⚠️镜头结尾（步骤⑥⑦，Roko重新掌控）：摄影机变得平稳、流畅、克制，只剩极细微、规律的呼吸式微动。
```

## 3. 焦段选择

| 用途 | 焦段 | 光圈 |
|---|---|---|
| 极致情绪大特写，额头到下巴填满画幅 | 85mm 或 100mm | F1.4 |
| 中景对白、双人镜头 | 50mm | F2.0-F2.8 |
| 广角建立、空间关系 | 35mm | F4-F5.6 |
| 道具或细节插入 | 50mm 或 85mm，焦点锁定物体 | F1.4 |
| 微距，毛孔、水滴、织物 | 45mm 微距 | F2.8 |

广角镜头要防止畸变：

```text
禁桶形畸变、禁枕形畸变、禁鱼眼效果、禁广角变形，画面线条必须笔直，构图平整。
```

浅景深插入和情绪特写要锁焦：

```text
⚠️焦平面严格锁定在[物体/角色]上——绝对禁止焦点漂移、禁止中途换焦、禁止自动跳焦。
```

## 4. 轨道、推进、拉远

慢速轨道或推进要写具体距离和时间：

```text
整个7秒内摄影机后移总距离仅约10-15厘米，速度慢到几乎察觉不到。禁变焦，禁突然推拉。
```

不要写“zoom”。写物理运动：推进、拉远、横移、升降、摇镜。

## 5. 镜头时长

| 镜头类型 | 建议时长 |
|---|---|
| 瞬间建立空间的广角闪现 | 0.3-0.5秒 |
| 一句中等长度对白 | 3-7秒 |
| 无台词反应，带情绪弧线 | 5-10秒 |
| 道具插入、远景、冻结 | 0.3-2秒 |
| 完整情绪大特写，5-7个表演节拍 | 8-15秒 |

如果一条提示词内有多个镜头，按这些时长内部拆分，并让内部镜头时长加总等于该提示词的预计时长。不要把总时长作为固定结尾写到提示词最后；把它写在 HTML 的时间安排列、提示词标题标签，或 `【镜头N｜起止秒数】` 内。

## 6. 常用模板

### 极致情绪大特写

```text
机位：85mm或100mm长焦，F1.4极浅景深，@image1严格极致大特写——额头到下巴填满画幅，脸部占据画面绝大部分。
摄影机运动：严格手持，全程有真实摄影师贴身跟拍的呼吸式微动。摄影机主动跟随@image1脸部细微运动：头部转动、抬起、下沉、目光移动时，构图同步微调。
摄影机位置：[角色东/西/南/北侧]。
背景：完全虚化为模糊色块和柔光斑，所有细节强烈散焦。
动作：[按编号写表演节拍]。
微表演细节：
- ① ...
```

### 道具插入镜头

```text
机位：50mm或85mm大光圈定焦，F1.4极浅景深。⚠️焦平面从第一帧到最后一帧严格锁定在[物体]上，绝对禁止焦点漂移。
清晰度规则：画面中只有[物体]绝对清晰锐利，其他元素全部强烈散焦为模糊色块。
摄影机角度：[角度]。整个X秒内摄影机总位移仅约10-15厘米，速度慢到几乎察觉不到。禁变焦，禁突然推拉。
```

### 广角空间闪现

```text
机位：35mm广角，全景/远景。⚠️持续时间严格约0.3-0.5秒，只用于瞬间建立空间，不是完整停留镜头。
动作：[关键动作]。⚠️0.3-0.5秒后立刻硬切到镜头2，无过渡、无淡出、无停留。
```

### 正上方俯拍冻结

```text
【镜头N】
机位：严格正上方俯拍。0.3-0.5秒冻结。
背景：严格匹配[场景]。
动作：所有人物位置冻结。时间静止。
```

## 7. 禁用运动

- 禁变焦：使用物理推进、拉远或横移。
- 禁稳定器：写手持就必须有手持呼吸感。
- 禁不受控抖动：除了有意的手持呼吸感，不能乱抖。
- 禁过度运动：亲密场景要让摄影机贴近人物，不要炫技。
