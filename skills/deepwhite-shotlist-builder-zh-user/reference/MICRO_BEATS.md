# 表演微节拍目录

不要在提示词里留下泛情绪。每个情绪都要拆成肌肉、呼吸、眼神、皮肤、姿态和时间节奏。这样提示词才像导演给演员的排练笔记，而不是模板化 AI 描述。

## 1. 基础规则

如果剧本或用户说“惊讶、紧张、悲伤、愤怒、害怕、动心、若有所思”等泛情绪，要先选择具体身体版本。

例如“惊讶”至少有四种：

- 轻微正向惊讶：眉毛轻抬，嘴唇微张，瞳孔短暂放大，柔和鼻吸气。
- 震惊：口腔急吸气，眼睛睁大，身体冻结，手不自主抬起。
- 难以置信：慢眨眼，头微倾，嘴唇压紧，单侧眉毛抬起。
- 惊喜：眼神亮起，笑意逐步建立，肩膀放松，一次轻松呼气。

不确定是哪一种时，问用户一个简短选择题。

## 2. 情绪目录

### 愤怒 / 决心 / 暴怒

```text
- 颊部咬肌透过皮肤可见地收紧或搏动
- 鼻翼在重音词上扩张
- 瞳孔收缩，视线聚焦变窄
- 眼角外缘收紧，但不要夸张眯眼
- 高潮时刻不眨眼
- 太阳穴或眉骨处出现细小汗珠
- 下颌向前顶出一瞬，又被压回去
```

### 焦虑 / 紧张

```text
- 喉结上下吞咽一次
- 台词前短促鼻吸气
- 舌尖舔润干燥下唇
- 下唇轻微内收
- 脸颊毛细血管轻微泛红
- 关键词时瞳孔放大
- 除非剧本明确写哭，否则不流泪
```

### 悲伤 / 脆弱（不落泪版）

```text
- 眼角外缘下垂
- 眼眶湿润、有眼神光，但泪水不落下
- 眉头中央皱起
- 嘴唇轻微颤抖一次
- 头部微微低垂
- 肩膀像失去支撑一样下沉
```

### 控制 / 冷静 / 优越感

```text
- 呼吸平稳均匀，和对手的紧张形成对比
- 手指和手臂放松
- 缓慢均匀眨眼
- 下颌微微抬起
- 微笑逐渐建立，眼角外侧出现细纹，不要一开始就是完成的笑
```

### 沉重 / 被压垮

```text
- 双肩下沉
- 头部微垂
- 深而慢的呼吸
- 声音比平时更低、更闷
- 回答时头部轻微侧倾5-15度
- 手臂无力垂在身体两侧
```

### 震惊 / 冻结

```text
- 镜头开始瞬间身体冻结0.3-0.5秒
- 冻结瞬间瞳孔放大
- 嘴唇微张但无声
- 冻结结束后一次延迟的急促鼻吸气
- 眼睛锁定触发源，不眨眼、不移开
```

### 压抑情绪

```text
- 他在试图忍住，面部肌肉都在对抗涌上来的情绪
- 颧骨处咬肌慢慢收缩
- 喉结一次延迟而用力的吞咽
- 眼眶积聚水光，但泪不落下
- 一次缓慢、深、控制的吸气，胸腔明显起伏
- 下颌微抖一次，立刻被收紧
```

## 3. 单镜头内的情绪弧线

情绪变化要写成编号节拍：

```text
①开始——[情绪A]——[具体肌肉/呼吸/眼神]
②触发——[触发事件]——[哪些肌肉释放或收紧]
③过渡——[情绪B开始出现]——[新的身体信号]
④...
⑦结尾——[最终情绪]——[最终身体状态]
```

同时写清贯穿整段必须可见的内容，例如眉头、下颌、呼吸、眼眶、视线锁定。

## 4. 对白和时间

每句台词都要有：

- 台词前动作：吞咽、吸气、舔唇、重心变化、视线移动。
- 台词中强调：哪个字或词被加重，鼻翼/瞳孔/下颌如何反应。
- 台词后停顿：保持视线、屏息、松开、移开、垂眼。

示例：

```text
台词前：一次短促鼻吸气，喉结吞咽。
台词：“别再问我。”——重音落在“再”，鼻翼在这个字上轻微扩张。
台词后：视线继续锁定对手0.5秒，然后微微向下移开。
```

## 5. 虚化背景中的倾听者

如果焦点人物清晰、其他人虚化，虚化人物也不能静止：

- 写清头和视线朝向
- 写清肩膀、头部、呼吸的微动作
- 对关键词做延迟反应
- 多人反应不要同步，错开0.3-0.5秒

## 6. 禁止写法

- 不要写“只是站在那里说话”
- 不要夸张鬼脸或卡通式表情
- 不要用“看向天花板”代表思考，必须写具体视线方向
- 不要默认流泪，除非剧本明确要求
- 多人不要完美同步反应
- 不要写“看起来很紧张/很开心/很难过”，必须拆成身体事件

## 7. 自检

交付前读一遍表演段落，问自己：这像不像导演看过演员排练后写下的具体笔记？如果读起来像通用模板，就重写。
