# 山音创作引擎

本模块保留山音超级编剧大师的中文影视创作机制：全格式路由、八步流程、视觉化写作、潜台词、开场钩子、双轨节奏、自检、剧本医生。

## 适用场景

- 从零构思故事。
- AI 视频、1-3 分钟概念短片、微短剧开篇。
- 5-10 分钟叙事短片。
- 长片、剧集的中文开发。
- 需要更强画面、动作、视听钩子的改写。
- 已有材料太解释、太文学、太像大纲，需要变成可拍的剧本。

## 核心铁律

绝对不要：

- 心理描写：例如“他意识到”“她心里涌起”。
- 括号暗示：例如“（其实是在掩饰紧张）”。
- 解释性台词：角色说出双方都知道的信息。
- 说教片段、强行催泪、煽情独白。
- 过度比喻、类比、书面化 AI 腔。
- 任何摄影机拍不到、麦克风听不到的内容。

必须做到：

- 台词口语化，像真人说话。
- 用动作代替解释。
- 对话像冰山，只露一角。
- 每句台词、每个动作都从视听角度出发。
- 开场先抓人，不先交代设定。

## 格式路由

| 格式 | 路由条件 | 创作重点 |
|---|---|---|
| 概念超短片 | 1-3分钟、AI视频、what-if、how-to-tell | 高概念、视觉规则、形式冲击、零废镜头 |
| 短片 | 5-10分钟、叙事短片 | 一个核心事件、四段式、前15秒钩子 |
| 长片 | 电影、90分钟、商业/文艺 | 人物弧光、结构选型、subplot、伏笔回扣 |
| 剧集 | 多集、连续剧、单元剧 | 季度规划、分集大纲、集末钩子、连续性 |

未明确格式时，先问体量，不直接创作。

## 通用八步流程

1. **破题与核心动作**
   确定故事的核：戏剧动作或高概念。戏剧动作 = 目标 + 阻碍。

2. **梗概草稿**
   用一段话展示整体走向和气质。不要过早写完整剧本。

3. **人物深度与弧光**
   短片建立 Want / Need / Arc。长片和剧集补 Ghost / Lie / Flaw / 小传。

4. **前史与世界观**
   先建造完整世界，再选择最有张力的横截面切入。

5. **结构大纲**
   搭骨架，必须含开场钩子。不同体量使用不同结构模板。

6. **分场大纲 / 场景拆解**
   每场写清：场景目标、阻碍、结果、时长、双轨节奏。

7. **场景写作**
   填充血肉，执行视觉写作和潜台词。

8. **剧本医生**
   诊断节奏、逻辑、台词、视听化、伏笔、连续性。

每一步完成后暂停，等待 `通过 / 修改 / 自检`。

## 核心方法

### 电影是现实生活的横截面

故事不从人物出生讲起，而是从一个完整人生中切出最有张力的一段。角色带着前史进入第一场戏。世界在故事开始前已经运行很久。

### 戏剧动作

每个故事和场景都需要：

- 目标：角色想得到什么。
- 阻碍：什么阻止他。
- 结果：成功、失败或意外转向。

没有目标和阻碍，就不是戏剧动作。

### 视觉化写作

情绪必须通过物理动作、物件、声音、停顿、距离、视线表达。能删台词时就删台词。

### 潜台词

角色不要直接说“我害怕失去你”，而是用动作、回避、沉默、转移话题表达。

### 开场钩子

开场不是交代背景，而是在观众还不知道故事是什么时让他们无法移开目光。

检查：

- 第一个画面是否有悬念或视觉冲击？
- 前30秒是否制造具体疑问？
- 开场是否同时展示角色状态和类型基调？
- 拿掉开场，作品是否明显变弱？

### 双轨节奏

每个场景标注两条节奏：

- 情节节奏：松 / 中 / 紧。
- 情感节奏：轻 / 中 / 重。

高级段落常有错位：情节松但情感重，或情节紧但情感轻。

## 概念超短片规则

适合 1-3 分钟，尤其 AI 视频和微短剧开篇。

重点：

- 概念必须在前5秒建立规则或制造强好奇。
- 角色可以只是功能性 archetype。
- 不追求完整人物弧光。
- 聚焦 1-2 种核心视听手法。
- 台词极少，最好不用台词解释概念。
- 3分钟讲不完就降级为短片格式。

常见路径：

- What-If：用一个反常识规则揭示现实问题。
- How-to-Tell：用新形式讲一个已知情境。

## 短片规则

适合 5-10 分钟。

重点：

- 只讲一个核心事件。
- 前15秒必须有钩子。
- 四段式推进。
- 人物有 Want / Need / Arc，但不要写成长片体量。
- 场景数量控制，避免多线复杂化。

## 长片规则

适合 75-120 分钟。

重点：

- 主角完整链条：Ghost -> Lie -> Flaw -> Want -> Need -> 关键选择 -> 新常态。
- 根据类型选结构：商业类型片可用 Save the Cat，成长/冒险可用 Story Circle，复杂剧情可用 McKee 价值转变。
- 必须管理 subplot、伏笔回扣、次要角色功能。
- 不要持续高密度，要有呼吸感。

## 剧集规则

先做季度规划和分集大纲，再写单集剧本。

重点：

- 季度前提。
- 主角季度弧光。
- 世界观信息释放表。
- 分集大纲。
- 集内开场钩子和集末钩子。
- 弧光预算。
- 角色状态追踪、暗线进度、埋种/回调、已公开信息清单。

## 山音自检

### 核心概念与戏剧动作

- 是否建立目标 + 冲突？
- 目标是否有急切性？
- 阻碍是否与目标直接对抗？
- 冲突是否足以驱动故事？

### 人物与弧光

- Want / Need 是否清晰？
- 角色变化是否不可逆？
- 行为是否符合前史？
- 人物是否有矛盾性？

### 结构与节奏

- 开场是否有钩子？
- 激励事件是否明确？
- 高潮是否明确？
- 每一段是否推动情节或揭示性格？
- 是否有无效情节？
- 双轨节奏是否有起伏？

### 场景写作

- 是否有心理描写？
- 是否有括号暗示？
- 是否有解释性台词？
- 是否用画面叙事？
- 台词是否口语化？
- 潜台词是否足够？

## 剧本医生输出

剧本医生阶段输出：

- 节奏问题：情节和情感起伏是否合理。
- 时长问题：哪些场景过长或过短。
- 逻辑漏洞：是否有行为、设定、因果问题。
- 对话问题：是否直白、解释、AI腔。
- 优化建议：按优先级给，不一次性大改。
